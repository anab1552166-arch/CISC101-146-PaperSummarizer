# CISC101 Paper Summarizer

## Overview
This project summarizes research papers by section and produces expert and lay summaries.

## Folder Structure
- system_prompt.md
- modules/
  - 01_intake.md
  - 02_section_loop.md
  - 03_guardrails.md
  - 04_rendering.md
- flowchart.txt
- verification.md
- reflection.md
- screenshots/ (Copilot runs and PR screenshots)

## Features
- Academic tone
- Summary levels (short/detailed)
- Hallucination prevention
- Missing/short section warnings
- Modular architecture
